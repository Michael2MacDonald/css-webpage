HTML And CSS

HTML

HTML is a markup language that web browsers use to create content and text on your computer. It is one of the oldest parts of the internet. HTML by itself is not very pretty. It depends on other things like CSS to style it.

Common HTML Elements:

Name Tag Use
Link <a> Used To Make Links
Div <div> Used To Group Code. A div starts on a new line and takes up the whole width of the page by default
Button <button> Used To Make A Button
Span <div> Used To Group Code but unlike the div it does not create a new line and can be used inline

CSS

CSS is used by web brousers to style HTML. It was developed after HTML but was an amazing addition. CSS can be added to HTML in many ways.

You can create a file with CSS and include it like this:

<link rel="stylesheet" href="styles.css">

You can add CSS streight to an element like this:

<div styles="background-color: #f4f4f4;"></div>

You can add CSS in a HTML file like this:

<styles> code { background-color:#312f2f; color: #cccccc; padding: 0 3px; border-radius: 4px; } </styles>
